EasyACT is a personal finance tracking web application designed to help you manage your daily income and expenses with intuitive interfaces and visual analytics. Take control of your financial goals with easy-to-use tools for recording transactions and analyzing your spending patterns.

## Technology Stack

EasyACT is built with modern, production-ready technologies:

### Frontend

- **Vue 3** (3.5.25) - Progressive JavaScript framework
- **PrimeVue** (4.5.3) - Rich UI component library
- **Tailwind CSS** (4.1.18) - Utility-first CSS framework
- **Chart.js** (4.5.1) - Interactive data visualization
- **Pinia** (3.0.4) - State management
- **Vite** (7.2.4) - Next-generation build tool

### Backend

- **Spring Boot** (4.0.1) - Enterprise Java framework
- **Java 25** - Latest Java runtime
- **Spring Data JPA** - Object-relational mapping
- **SpringDoc OpenAPI** (2.6.0) - Automatic API documentation

### Database

- **PostgreSQL 15** - Robust relational database

## Architecture Overview

EasyACT uses a containerized microservices architecture for easy deployment and scalability:

```
┌─────────────┐      ┌─────────────┐      ┌─────────────┐
│             │      │             │      │             │
│   Web UI    │─────▶│   API       │─────▶│  Database   │
│  (Vue.js)   │      │ (Spring     │      │ (PostgreSQL)│
│   Nginx     │      │  Boot)      │      │             │
│             │      │             │      │             │
└─────────────┘      └─────────────┘      └─────────────┘
   Port: 8000           Port: 8080          Port: 5432
```

<Note>
All three services run in Docker containers and communicate through a bridge network, making deployment simple and consistent across environments.
</Note>
